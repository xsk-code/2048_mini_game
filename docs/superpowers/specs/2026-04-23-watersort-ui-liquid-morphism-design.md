# 水排序 UI 拟态化重设计方案

**日期：** 2026-04-23

## 1. 目标

对水排序游戏的 UI 进行拟态化重设计，使液体更像真实液体，试管更像真实玻璃器皿，布局更有层次感。

## 2. 当前 UI 问题诊断

通过阅读 `game.js` 中 `drawWatersortTube` 和 `recalculateWatersortLayout` 的代码，当前 UI 存在以下问题：

| 问题 | 现状 | 影响 |
|------|------|------|
| 液体过于"方块化" | 每层液体用 `ctx.rect` 绘制矩形，仅最底层有圆弧 | 看起来像彩色积木，不像液体 |
| 液体无表面张力 | 顶部液体是一条平直线 | 缺少真实液体的弯月面/弧面效果 |
| 无液面波动 | 液体完全静止，无任何动态暗示 | 感觉像固体色块 |
| 试管造型简陋 | 仅用线条描边 + 半透明填充 | 缺少玻璃质感、折射、高光 |
| 5 瓶换行显示 | `maxCols = Math.min(tubeCount, 4)` 硬编码最多 4 列 | 5 个瓶子被迫换行，浪费空间 |
| 选中反馈弱 | 仅上移 20rpx | 缺少发光/弹跳等生动反馈 |

## 3. 设计方向：「实验室玻璃器皿」美学

**核心概念**：将游戏画面想象为一个精致的化学实验室桌面——试管是真正的玻璃器皿，液体有真实的物理质感（弯月面、光泽、折射）。

**关键词**：玻璃质感 · 弯月面 · 表面张力 · 柔和光晕

## 4. 布局重设计

### 4.1 5 瓶同行显示

将 `maxCols` 从 4 改为动态计算，5 瓶及以下全部同行。

```
tubeCount ≤ 7  →  全部同行
tubeCount = 8  →  5+3 双行
tubeCount = 9  →  5+4 双行
tubeCount = 10 →  5+5 双行
tubeCount = 11 →  6+5 双行
tubeCount = 12 →  6+6 双行
```

**尺寸自适应**：
- 5 瓶同行时，单管宽度 = `(可用宽度 - 4 × gap) / 5`
- 管宽范围：`rpx(56) ~ rpx(72)`，根据数量自动缩放
- 管高 = 管宽 × 3.0

当试管数 > 7 需要双行时，两行试管在各自行内保持顶部齐平。

## 5. 试管造型重设计

### 5.1 玻璃试管外形

当前试管只是简单的线条描边。新方案将试管绘制为真正的玻璃容器：

```
     ╭────────╮    ← 管口：略宽于管身，有翻边
     │  玻璃  │    ← 管身：微微内收的梯形（上宽下窄）
     │  壁面  │
     │  高光  │    ← 左侧高光条：模拟玻璃折射
     │        │
     ╰────────╯    ← 底部：圆弧底（半圆）
```

**绘制要点**：
1. **管口翻边**：顶部画一个略宽的椭圆弧，模拟玻璃管口的厚度
2. **管身微锥**：管身从上到下宽度递减约 `rpx(4)`，模拟真实试管的锥度
3. **底部圆弧**：用半圆弧代替当前的 quadraticCurveTo
4. **玻璃壁厚**：用双层路径——外层深色、内层浅色，表现壁厚

### 5.2 玻璃质感渲染

| 层次 | 绘制方式 | 效果 |
|------|----------|------|
| 外壁 | 半透明白色描边 `rgba(255,255,255,0.3)` | 玻璃边缘光 |
| 内壁 | 半透明深色描边 `rgba(0,0,0,0.05)` | 内侧阴影 |
| 左侧高光条 | `rgba(255,255,255,0.25)` 竖直矩形 + 高斯模糊 | 玻璃折射 |
| 右侧反光 | `rgba(255,255,255,0.08)` 窄竖条 | 环境光反射 |
| 底部阴影 | 管底下方 `rpx(4)` 处的模糊椭圆 | 投影感 |

## 6. 液体拟态化重设计

这是最核心的改动——让液体看起来像真正的液体。

### 6.1 弯月面（Meniscus）效果

真实液体在玻璃容器中，因表面张力会形成凹形弯月面——中间低、两侧高。

**绘制方法**：
- 液体顶层不再用直线，而是用二次贝塞尔曲线绘制凹弧
- 弧度参数：控制点 Y 偏移 = `liquidHeight × 0.3`（凹下去约液层高度的 30%）

```
当前：  ──────────   （平直线）

新方案： ╰────────╯   （凹形弯月面）
```

**代码示意**：
```javascript
ctx.beginPath();
ctx.moveTo(liquidLeft, liquidY + meniscusHeight);
ctx.quadraticCurveTo(
  liquidCenterX, liquidY,
  liquidRight, liquidY + meniscusHeight
);
ctx.lineTo(liquidRight, liquidBottom);
ctx.lineTo(liquidLeft, liquidBottom);
ctx.closePath();
```

### 6.2 液体渐变增强

当前液体只有简单的上下 `light → dark` 渐变。新方案增加左右渐变模拟光线折射：

| 渐变方向 | 效果 |
|----------|------|
| 上→下 | `light → dark`（保持，模拟光照） |
| 左→右 | 左侧加一层 `rgba(255,255,255,0.12)` 的白色叠加（模拟玻璃折射增亮） |

**实现**：先画基础上下渐变，再叠加一层左右渐变的半透明白色。

### 6.3 液体表面高光

在弯月面弧线上方，画一条细弧形高光线：

- 颜色：`rgba(255,255,255,0.5)`
- 宽度：`rpx(1.5)`
- 位置：沿弯月面曲线，偏移 `rpx(1)` 上方
- 效果：模拟液面反光

### 6.4 液体内部气泡（可选增强）

在液体内部随机位置绘制 2-3 个微小圆形气泡：

- 半径：`rpx(1.5) ~ rpx(3)`
- 颜色：`rgba(255,255,255,0.2)`
- 位置：基于 `tubeIndex + layer` 的伪随机，确保每帧一致
- 效果：增加液体的"活性"感

### 6.5 液体层间融合

当前每层液体之间有明显的分界线。新方案在相邻同色液体层之间消除分界线，使其融为一体：

```javascript
// 如果当前层与上一层颜色相同，则当前层顶部不画分界线
// 直接从上一层底部开始填充，形成连续的液体柱
```

## 7. 交互反馈增强

### 7.1 选中试管效果

当前选中仅上移 20rpx，新方案增加：

| 效果 | 实现 |
|------|------|
| 弹跳上移 | 上移 `rpx(24)` + 缓动动画 |
| 管口发光 | 管口处画一圈 `rgba(255,255,255,0.3)` 的光晕 |
| 液体晃动 | 选中时弯月面振幅增大（凹得更深），暗示"准备倒出" |

### 7.2 倒水动画增强

当前倒水动画仅使用 `watersortTubeOffsets` 做简单的 Y 轴偏移。新方案：

1. **源管**：液体从顶部逐层消失，弯月面下移
2. **目标管**：液体从顶部逐层出现，弯月面上升并带有"溅入"的微波动
3. **倒出弧线**：在两个管口之间画一条抛物线水柱（可选）

### 7.3 完成管闪烁

当一管完成排序时：
- 液体整体发出柔和脉冲光（`opacity` 在 0.8~1.0 间缓动）
- ✓ 标记改为液体上方浮出的气泡式 ✓

## 8. 配色方案调整

### 8.1 试管配色

| 元素 | Light 模式 | Dark 模式 |
|------|-----------|-----------|
| 管壁外层 | `rgba(200, 215, 230, 0.5)` | `rgba(120, 150, 180, 0.35)` |
| 管壁内层 | `rgba(240, 245, 250, 0.2)` | `rgba(30, 50, 80, 0.2)` |
| 左侧高光 | `rgba(255, 255, 255, 0.3)` | `rgba(255, 255, 255, 0.15)` |
| 管口翻边 | `rgba(180, 200, 220, 0.7)` | `rgba(100, 130, 170, 0.5)` |
| 底部投影 | `rgba(0, 0, 0, 0.06)` | `rgba(0, 0, 0, 0.2)` |

### 8.2 液体配色（保持现有色系，增加折射层）

每色液体由 3 层渐变叠加：
1. **基础层**：`light → dark` 纵向渐变（现有）
2. **折射层**：左侧 `rgba(255,255,255,0.12)` → 右侧透明的横向渐变
3. **弯月面高光**：顶部弧线 `rgba(255,255,255,0.45)`


### 9.2 `drawWatersortTube()` 改动

```
1. 绘制管底投影（模糊椭圆）
2. 绘制试管外壁路径（梯形 + 圆弧底）
3. 填充管内背景（半透明）
4. 绘制液体（从底到顶）：
   a. 判断与上层是否同色 → 融合/分隔
   b. 顶层液体：弯月面路径（quadraticCurveTo 凹弧）
   c. 非顶层液体：矩形路径
   d. 填充基础纵向渐变
   e. 叠加横向折射渐变
   f. 顶层弯月面高光线
   g. 气泡（可选）
5. 绘制玻璃高光条（左侧竖条）
6. 绘制管口翻边
7. 绘制选中效果（光晕/弹跳）
8. 绘制完成标记
```

## 11. 优先级与实施建议

| 优先级 | 改动项 | 影响程度 | 复杂度 |
|--------|--------|----------|--------|
| P0 | 5瓶同行显示 | ⭐⭐⭐⭐⭐ | 低 |
| P0 | 弯月面效果 | ⭐⭐⭐⭐⭐ | 中 |
| P1 | 玻璃质感（高光+折射） | ⭐⭐⭐⭐ | 中 |
| P1 | 液体层间融合 | ⭐⭐⭐⭐ | 低 |
| P2 | 管口翻边 + 管底投影 | ⭐⭐⭐ | 低 |
| P2 | 选中光晕增强 | ⭐⭐⭐ | 低 |
| P3 | 液体气泡 | ⭐⭐ | 低 |

建议按 P0 → P1 → P2 → P3 顺序实施，每完成一个优先级即可看到明显的视觉提升。
